## ¿Qué es Markdown?
Markdown es un lenguaje de marcado ligero que se utiliza para dar formato a texto de manera sencilla.  
Se usa comúnmente para crear archivos README, documentación, blogs y notas técnicas, ya que permite estructurar contenido con encabezados, listas, enlaces, código y otros elementos de forma clara y rápida sin necesidad de usar HTML.
